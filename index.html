TEXTURE PACK ULTIMATE TUTORIAL

how to make a Minecraft texture pack

This is a tutorial on how to make a Minecraft texture pack, follow these steps to make your very own texture pack for your Minecraft world

STEP.1

Go to versions file of .minecraft,
then copy the jar of the version you want to make texture pack of, copy it, and paste it to a blank folder.

STEP.2

Next, extract the files, it should take some time, so sit down and relax.
after extracting the files, go in and copy assets, then delete all the other files, then paste assets into the folder.

STEP.3

Go into the assets folder, Minecraft, texture, then choose the thing that you want to change,
I will choose blocks here, select any block you like, then open up paint 3D

STEP.4

paint any pattern you like on the block!!
then press save, go to assets, then create a text file called pack.

STEP.5

type the following word in the file











{

"pack": {

"pack_format": 6,

"description": "[Any description you want to give your pack]"

}

}

STEP.6

the texture pack version is listed down below

pack_format Minecraft version
1 1.6.1 – 1.8.9
2 1.9 – 1.10.2
3 1.11 – 1.12.2
4 1.13 – 1.14.4
5 1.15 – 1.16.1
6 1.16.2 – 1.16.5
7 1.17 - 1.17.1
8 1.18.x
9 1.19.x

STEP.7

click save as new file
change the sub-name of the file to all files, then add .mcmeta at the back of the name.

STEP.8

zip the two files which is assets and pack.mcmeta
the your pack is finished! You can share to your friends or upload to the internet, have fun making texture packs!
